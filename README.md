# CodeAlpha_StockTradingPlatform-
A simple Java console application that simulates a basic stock trading platform. Users can view available stocks, buy and sell shares, track their portfolio value, and manage account balance. This project demonstrates OOP concepts, collections, and menu-driven programs in Java.


#  Stock Trading Application (Java)

##  Overview

This is a **console-based Stock Trading Application** developed using **Core Java**.
The application allows users to simulate stock market operations such as buying, selling, and tracking stocks.

It is designed for **beginners** to understand:

* Object-Oriented Programming (OOP)
* Java Collections (`ArrayList`, `HashMap`)
* Menu-driven programs
* Real-world application structure

---

##  Technologies Used

* Java (Core Java)
* Scanner class for user input
* OOP concepts (Encapsulation, Abstraction)
* Collections Framework

---

# Features

* View available market stocks
* Buy stocks based on available balance
* Sell owned stocks
* View portfolio with total value
* Track remaining balance
* Menu-driven interactive console UI

---

# How to Run

1. Clone the repository
2. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Run `Main.java`
4. Follow on-screen menu options
